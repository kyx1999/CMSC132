Specifications

  • For this project you will implement classes that allow us to generate HTML5. A description of the classes you need to implement can be found at javadoc.
  • In a web page we can have text (represented by the TextElement class) or tags (represented by the TagElement class). Both classes implement the Element interface that defines a method (genHTML) that generates the expected HTML for the element. Tag element classes extend the TagElement class.
  • The WebPage class represents a web page and will put together all the different elements. Notice this class implements the Comparable interface. For simplicity, the tags associated with doctype, html, head, meta, and body will not be generated using classes. One of the method asks to create a file with HTML code. We have provide a method that allows you to create such a file (see Utilities class).
  • Try to generate HTML that has reasonable indentation, but keep in mind the submit server ignores it.
  • Do not use the System.lineSeparator() method. It creates a newline that will not allow you to pass tests.
  • For the <a> tag any attributes go after the href attribute. If you add them before you will not pass tests.
  • HTML information can be found at IntroHTML.pdf and IntroHTMLCode.zip.
  • The best way to understand what you need to implement is to study the public tests.
